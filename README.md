🚗 Elite Auto Care (Car Detailing Website)

Elite Auto Care is a car detailing and car care service website where customers can explore services, book appointments, and learn more about vehicle maintenance. The website provides services like interior cleaning, exterior polishing, ceramic coating, paint protection, engine cleaning, and premium car care solutions.

🚀** Technologies Used**
**Frontend**
**HTML5** – Creates the structure of the website.
**CSS3** – Makes the website attractive and responsive.
**JavaScript** – Adds interactive features and functionality.
**Website Features**
Service showcase section
Online booking system
Customer reviews and testimonials
Customer login portal
Admin dashboard for managing services and bookings
**Backend**
**Node.js** – Runs the server.
**Express.js** – Helps build APIs and handle requests.
**JWT Authentication** – Provides secure user login.
**bcrypt** – Encrypts passwords for better security.
**CORS & Dotenv** – Used for API access and environment settings.
Database
**SQLite** – Stores user details, service information, and booking records.
**🛠️ Tools Used**
**npm** – Manages project packages and dependencies.
**nodemon** – Automatically restarts the server when changes are made.
